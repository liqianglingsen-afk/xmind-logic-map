# XMind Logic Map

A Codex plugin that turns videos, documents, web pages, transcripts, and long-form text into reliable, editable XMind mind maps.

It includes source links, structured notes, relevant screenshots when appropriate, and a validator for the JSON-based XMind package format.

## Included skill

`xmind-logic-map` creates an `.xmind` file and verifies ZIP integrity, JSON structure, notes, IDs, links, and image resources before handoff.

## Local installation

The plugin is registered in the local `personal` Codex Marketplace. Install or refresh it with:

```powershell
codex plugin add xmind-logic-map@personal
```

Start a new Codex task after installation so the skill is available.
